# LandDesk CRM 🏡
A land wholesaling CRM with pipeline tracking, reminders, and deal calculator.

## Features
- **Pipeline** — Kanban board across all deal stages
- **Table** — Spreadsheet view of all leads
- **Reminders** — Due today / overdue alerts, linked to leads
- **Deal Calculator** — MAO formula calculator
- **Auto-save** — Everything saves to your browser automatically

## Deploy to Vercel (Free — 5 steps)

### Step 1 — Install Node.js
Download from https://nodejs.org (click "LTS" version) and install it.

### Step 2 — Create GitHub account
Go to https://github.com and sign up (free).

### Step 3 — Upload this folder to GitHub
1. Click **+** → **New repository** → name it `landdesk-crm`
2. Set to **Private** → click **Create repository**
3. Click **"uploading an existing file"**
4. Drag ALL the files from this zip into the upload area
5. Click **Commit changes**

### Step 4 — Deploy on Vercel
1. Go to https://vercel.com → Sign up with GitHub
2. Click **"Add New Project"**
3. Select your `landdesk-crm` repo
4. Leave all settings as default → click **Deploy**
5. ✅ Done! You'll get a live URL like `landdesk-crm.vercel.app`

### Step 5 — Access anywhere
Bookmark your Vercel URL. Open it on any device — your data saves in the browser.
